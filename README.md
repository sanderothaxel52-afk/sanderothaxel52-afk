# Steal the AI

A browser game about building a base, buying AIs, running your operation, and stealing stronger AIs while keeping heat under control.

## Play on GitHub Pages

If this repository is named `sanderothaxel52-afk`, the site URL is:

`https://sanderothaxel52-afk.github.io/sanderothaxel52-afk/`

If the repository is later renamed to `sanderothaxel52-afk.github.io`, the site URL becomes:

`https://sanderothaxel52-afk.github.io/`

## Enable Pages

1. Open repository **Settings**
2. Open **Pages**
3. Under **Build and deployment**:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/ (root)**
4. Save

## Files

- `index.html` - UI
- `styles.css` - styling
- `script.js` - game logic

## Gameplay

- Buy AIs from the market
- Run days to earn money
- Upgrade your base for more capacity
- Steal stronger AIs through risky raids
- Keep heat under control or enforcement will punish you
